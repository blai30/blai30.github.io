A weather dashboard built with [Vite](https://vite.dev), [Preact](https://preactjs.com), and Tailwind CSS. It fetches current conditions and the forecast from [Open-Meteo](https://open-meteo.com), which is free and does not require an API key, and presents everything in a fixed 16:9 layout meant to sit on a wall display or second monitor.

The dashboard is condition-reactive: the background and accents change based on the weather and the time of day, so a clear afternoon, a rainy evening, and a thunderstorm at night each get their own distinct look.
