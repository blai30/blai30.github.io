BaySpace was an image posting web application built with a team for a CS 648 course project. The frontend used Bootstrap with plain HTML, CSS, and JavaScript, while the backend ran on Node.js and Express against a MySQL database. It included user registration and authentication, search and filters, and responsive interface. Developed by a team of 8. It was originally hosted on an AWS EC2 instance running Ubuntu.

The app is now archived. The original AWS instance it ran on has been permanently deleted, so the live site is no longer available, but the source remains as a record of the work.
