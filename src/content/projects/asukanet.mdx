AsukaNet is a Discord bot built on .NET with the [Discord.Net](https://discordnet.dev) library. It groups its features into clear categories: fun commands, music playback, role assignment, and general server utilities. The bot runs from a single console application, with no communication to any backend API or database. Role management is handled using unique string keys based on the role snowflake id, which means no database is needed to keep track of which roles are which.
