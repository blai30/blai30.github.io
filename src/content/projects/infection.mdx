Infection is a first-person shooter built in [Unity](https://unity.com). Multiplayer is handled by [Mirror](https://mirror-networking.com), which manages the networked state between players so matches can run across multiple clients.

Matches are round-based and allow up to 8 players with weapons located around the map for players to wield. It features custom shaders like a scope shader that I created myself, particle VFX, and 3D audio with FMOD. Developed by a team of 8.
